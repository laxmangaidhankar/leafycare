# 🌱 Smart Plant Care & AI Plant Health Assistant

An intelligent Android application designed to help users manage, monitor, and improve plant health using AI-based recommendations, reminders, and smart gardening insights.

---

## 📱 Project Overview

This application is built to assist plant owners in maintaining healthy plants through:
- AI-powered plant disease detection
- Personalized care recommendations
- Smart reminders for watering and maintenance
- User-friendly onboarding and setup
- Google authentication and secure login system

The goal is to make plant care **simple, intelligent, and accessible for everyone**.

---

## 🚀 Features

### 🌿 Core Features
- Splash Screen with branding
- Get Started onboarding flow
- User Authentication (Sign In / Sign Up)
- Google Sign-In integration
- Plant preference questionnaire (Garden / Potted plants)
- Personalized home dashboard
- Plant care reminders

### 🤖 Smart Features
- AI-based plant health suggestions (conceptual/extendable)

- Smart user profiling based on preferences
